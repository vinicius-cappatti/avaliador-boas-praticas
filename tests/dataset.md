# Dataset utilizado

Este dataset consiste de 25 arquivos de código-fonte na linguagem Python, gerados via Claude Opus 4.6. Do total de arquivos:
- 13 arquivos estão escritos seguindo perfeitamente as diretrizes de melhores práticas da PEP 8.
- 12 arquivos contêm violações propositais de espaçamento, indentação e quebra de linhas para validar o funcionamento do agente avaliador.

Os arquivos estão localizados no diretório auxiliar, com os resultados previstos mapeados na pasta `gabarito`.
